# Rough Program (Quick Start)

## Requirements
- Python 3.9+
- Install deps: pip install -r requirements.txt

## Run
- python rough_program.py
- Select COM port, duration, confirm start

## Data / Viewing
- Raw PPG saved to [ppg_data.txt](ppg_data.txt)
- Window Data saved to [results.txt](results.txt)
- Live view: run [plot_realtime_data.py](plot_realtime_data.py)

## Erica instructions for your messed up python
If you run into errors:
- Ctrl + Shift + P and search for select: python interpreter.
- Choose the venv in the python_code folder. In your cmd terminal activate it with ```venv\Scripts\activate```
